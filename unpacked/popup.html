使用说明

  • 此插件只能工作于devtool环境下, 他可以读取整站的资源信息. 你需要刷新你的浏览器 或者重新打开devtool面板, 然后切换到Jiayou_web 标签.
  • 点击"下载/保存所有资源"按钮，整站资源将保存到本地.
  • 勾选"下载所有域名下的资源"将会下载所有域名下的资源.
  • 勾选 "下载资源使用zip压缩包" , 所有资源将被打进压缩包里 (<DOMAIN>.zip).
  • 如果你想手动保存文件, enable option from Chrome > Settings > Downloads > Ask where to save each file before downloading
